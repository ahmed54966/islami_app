🕌 Islami App - Flutter Application
A comprehensive Islamic application built with Flutter designed to provide a rich spiritual experience. The app features a clean, modern UI with support for both Light and Dark modes, and is fully localized to support multiple languages.

✨ Features
Holy Quran: Browse all Surahs with a clear reading view for verses (Ayat).

Hadith Section: Access a collection of Islamic Hadiths with a dedicated detailed view.

Islamic Radio: Listen to various Quranic radio stations (Live Stream integration).

Electronic Sebha: Digital tasbih counter with an interactive and animated interface.

Theming: Full support for Light Mode and Dark Mode with high-quality assets.

Localization: Easily switch between Arabic and English languages.

🚀 Tech Stack & Concepts
Framework: Flutter (Dart)

State Management: Provider.

Assets Management: Custom fonts (El Messiri) and high-resolution Islamic vectors.

Localization: Flutter Localizations (Intl).

UI/UX: Custom TabBar navigation and themed widgets.
